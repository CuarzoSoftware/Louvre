Detailed Description

Keyboard modifiers event.

Keyboard modifiers events are automatically sent to client surfaces when they acquire keyboard focus.

Class Documentation

◆ Louvre::LKeyboardModifiersEvent::Modifiers

struct Louvre::LKeyboardModifiersEvent::Modifiers

Keyboard modifiers.

Class Members
UInt32 depressed = 0 Active modifiers when physically pressed.
UInt32 latched = 0 Hooked modifiers that will be disabled after a non-modifier key is pressed.
UInt32 locked = 0 Active modifiers until they are pressed again (e.g. the Shift key)
UInt32 group = 0 Group the above states (use this value if the source of a modifier change is not of your interest)
